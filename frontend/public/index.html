Notion Dashboard

Báo cáo & phân tích dữ liệu

Hướng dẫn lấy báo cáo

1 Chọn dự án ở sidebar trái
2 Chọn loại báo cáo từ dropdown
3 Nhấn "Tạo Báo Cáo"

Phím tắt

ESC Đóng popup/modal
Ctrl/Cmd+F Focus ô tìm kiếm sidebar
Ctrl/Cmd+Enter Tạo báo cáo ngay
Ctrl/Cmd+Shift+R Làm mới dữ liệu
Ctrl/Cmd+Shift+A Chọn tất cả dự án hiển thị
Ctrl/Cmd+Shift+X Bỏ chọn tất cả
Ctrl/Cmd+Shift+H Ẩn/Hiện mục đã ẩn
Ctrl/Cmd+S Lưu cấu hình sidebar
Alt+1..5 Chọn nhanh loại báo cáo
Ctrl/Cmd+D Fill down (bảng Năng suất)

Các loại báo cáo

Sprint Report
  • Theo dõi tiến độ sprint
  • Biểu đồ phân bổ theo Status
  • Thống kê theo Assignee
Productivity Report
  • Đo lường năng suất làm việc
  • So sánh giữa các thành viên
  • Phân tích xu hướng theo thời gian
Raw Data Report
  • Xem toàn bộ dữ liệu chi tiết
  • Lọc theo thời gian và status
  • Xuất file Excel đầy đủ
Raw All Projects
  • Gộp tất cả Task database
  • Chuẩn hóa cột tự động
  • Dashboard phân tích tổng quan
Burndown Chart
  • Biểu đồ tiến độ Sprint
  • So sánh Ideal vs Actual
  • Theo dõi điểm còn lại
